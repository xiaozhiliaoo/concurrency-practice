# concurrency-practice

Java并发学习

这是Java并发研究的书籍和示例集合，还有源码姊妹篇源码分析集合。研究基于**操作系统层，语言层，应用层**。

**操作系统层**主要参考进程管理(进程调度，同步)：《深入理解计算机系统》，《操作系统概念》

**语言层**主要依赖于Java。

**应用层**主要是仓库示例。
