##Java并发学习

这是java并发书籍和示例集合，还有源码姊妹篇源码分析集合 。

研究基于操作系统层，语言层，应用层。

操作系统层主要参考进程管理(进程调度，同步)：深入理解计算机系统，操作系统概念
语言层主要依赖于Java。
